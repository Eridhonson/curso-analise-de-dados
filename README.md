# análise de dados
Atividades complementares do curso Introdução à Análise de Dados para pesquisa no SUS.
Os arquivos de dados (.cvs, .xlsx e .parquet)  e as instruções das atividades estão no repositório https://github.com/CampusVirtualFiocruz/curso-analise-de-dados
Do repositório fonte pegarei apenas os dados e as instruções, os scripts R serão criados e commitados na minha máquina local, e isso inclui:
- Os resultados,
- os processos para a obtenção dos resultados, e
- os inúmeros erros que serão cometidos no processo.
